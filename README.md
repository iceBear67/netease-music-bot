# NCM Resolver

网易云音乐下载器  
![](https://upload.cc/i1/2022/07/10/Bsno7U.png)

不想自己搭的可以用 [@netease_music_bot](https://t.me/netease_music_bot)

# Commands

`/ncm`

# Setup

修改 `config.py` 然后直接运行 `bot.py` 即可 (Python 3.9+)  
要能连接到 Telegram & 网易云
